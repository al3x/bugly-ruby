h1. Introduction

This is a Ruby library for accessing the "Bugly":http://bug.ly REST API.

h1. Supported Resources

The library currently supports *reading* all documented API-accessible resources:

* Issues
* Projects
* Milestones
* Categories
* Labels
* Views
* Users
